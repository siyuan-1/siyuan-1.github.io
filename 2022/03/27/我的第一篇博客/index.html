伪数组并不是真正的数组
具有数组的length属性
按照索引的方式进行储存的
它没有真正数组的一些方法pop（） push（） 等等

    function geiMax() {
        var max = arguments[0]
        for (var i = 1; i < arguments.length; i++) {
            if (arguments[i] > max) {
                max = arguments[i]
            }
        }
        return max;
    }
